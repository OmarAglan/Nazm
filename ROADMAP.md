# خارطة طريق نَظْم

هذه الخارطة تقسم العمل إلى مراحل صغيرة قابلة للتحقق. كل مهمة يجب أن تنتهي بتغيير محدود، اختبار مناسب، وتحديث وثائقي عند الحاجة.

آخر تحديث: 2026-06-29، بعد إغلاق بوابة صحة الترميز والمخرجات 7.1 باختبارات مصفوفية وتفاضلية وحدود صريحة وسياسة Unicode مثبتة. الأولوية الحالية انتقلت إلى مهام تجربة المستخدم 7.2؛ ولا يعني إغلاق البوابة أن تغطية باء أو قبول الروابط الخارجية اكتملت.

- [x] اختبار البحث عن وسم موجود: تأكيد أن العنوان المحسوب يعود كما هو.
- [x] اختبار البحث عن وسم مفقود: تشخيص عربي عند مرجع غير محلول.
- [x] حساب أحجام التعليمات الثابتة: تأكيد تعليمات مثل `ret` و`nop` و`syscall`.
- [x] حساب أحجام التعليمات ذات الفوريّات: ضمان اختيار حجم فوري مناسب أو رفض غير المدعوم.

- [ ] توثيق عقد المرور الأول: ما الذي يحسبه وما الذي لا يغيره.

## المرحلة 5: ترميز x86-64 الأساسي

- [x] توسيع اختبارات REX: تغطية `r8` إلى `r15` في كل مواقع ModRM المهمة.
- [x] اختبار ModRM للسجل إلى سجل: حالات المصدر والوجهة الأساسية والممتدة.
- [x] اختبار ModRM للذاكرة: base register، displacement 8-bit، وdisplacement 32-bit.
- [x] اختبار SIB عند الحاجة: خصوصا `rsp` و`r12` لأنهما يحتاجان SIB حتى في الحالات البسيطة.
- [x] اختبار ترميز `mov`: سجل/فوري، سجل/سجل، ذاكرة/سجل، وسجل/ذاكرة.
- [x] اختبار ترميز الحسابيات: `add`, `sub`, `imul`, `idiv`, `inc`, `dec`, و`neg`.
- [x] اختبار ترميز المنطق: `and`, `or`, `xor`, `not`, وعمليات الإزاحة.
- [x] اختبار ترميز المقارنة والقفز: `cmp`, `test`, والقفزات الشرطية الأساسية.
- [x] وضع حد صارم لطول التعليمة: تأكيد عدم تجاوز 15 بايت في كل نتيجة ترميز.
- [x] تقسيم جدول الترميز عند الحاجة: إبقاء الإضافة سهلة مع نمو عدد التعليمات.
