# أمثلة نَظْم

تحتوي هذه المجلدات على مجموعة صغيرة مقصودة من الأمثلة ذات الأسماء العربية
الوصفية:

- `مرحبا.نظم`: يكتب رسالة UTF-8 عبر نداء نظام لينكس.
- `خروج.نظم`: أصغر ملف قابل للربط والخروج بالرمز صفر.
- `حلقة.نظم`: يوضح المقارنة والقفز الشرطي واستخدام `سجل_العداد`.
- `بيانات.نظم`: يغطي توجيهات البيانات والحجز الصفري وقيد ترحيل إلى وسم بيانات.
- `diagnostics/`: حالات خطأ قصيرة مقصودة لاختبار التشخيصات.

لتجميع مثال إلى إلف64:

```bash
نظم --صيغة إلف64 --خرج build/مثال.o examples/خروج.نظم
```

ولتجميعه إلى كوف:

```bash
نظم --صيغة كوف --خرج build/مثال.obj examples/خروج.نظم
```

لا تعد ملفات `diagnostics/` أمثلة ناجحة، ولا ينبغي إدخالها في اختبار تجميع
الأمثلة الجيدة.
