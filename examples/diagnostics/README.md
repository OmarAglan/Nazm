# أمثلة التشخيصات العربية

هذه الملفات مقصودة الخطأ. الهدف منها إظهار شكل رسائل نَظْم العربية مع اسم الملف، السطر، العمود، سطر المصدر، ومؤشر `^ هنا`.

شغّل أي مثال بهذا الشكل:

```bash
./nazm examples/diagnostics/فاصلة-مفقودة.مجمع
```

الملفات:

- `فاصلة-مفقودة.مجمع`: تعليمة بمعاملين دون الفاصلة العربية `،`.
- `محرف-غير-معروف.مجمع`: محرف لا يقبله المحلل المعجمي.
- `وسم-مكرر.مجمع`: تعريف الوسم نفسه مرتين.
- `وسم-غير-محلول.مجمع`: قفز إلى وسم غير معرّف.
- `ذاكرة-غير-صالحة.مجمع`: معامل ذاكرة ناقص داخل الأقواس.

أخطاء شائعة أخرى مثبتة في
[`Docs/LANGUAGE.md`](../../Docs/LANGUAGE.md#أخطاء-شائعة): الحركات المطلوبة في
`نادِ` و`انفِ`، استخدام أرقام ASCII داخل أسماء السجلات، حدود الإزاحات
والفوريّات، تعارض `.عام` و`.محلي`، وتسلسلات هروب السلاسل المقبولة.
