# أمثلة التشخيصات العربية

هذه الملفات مقصودة الخطأ. الهدف منها إظهار شكل رسائل نَظْم العربية مع اسم الملف، السطر، العمود، سطر المصدر، ومؤشر `^ هنا`.

شغّل أي مثال بهذا الشكل:

```bash
./نظم examples/diagnostics/فاصلة-مفقودة.نظم
```

الملفات:

- `فاصلة-مفقودة.نظم`: تعليمة بمعاملين دون الفاصلة العربية `،`.
- `محرف-غير-معروف.نظم`: محرف لا يقبله المحلل المعجمي.
- `وسم-مكرر.نظم`: تعريف الوسم نفسه مرتين.
- `وسم-غير-محلول.نظم`: قفز إلى وسم غير معرّف.
- `ذاكرة-غير-صالحة.نظم`: معامل ذاكرة ناقص داخل الأقواس.

أخطاء شائعة أخرى مثبتة في
[`Docs/LANGUAGE.md`](../../Docs/LANGUAGE.md#أخطاء-شائعة): تهجئات 0.3 المحذوفة،
أسماء السجلات الوصفية، حدود الإزاحات والقيم الفورية، تعارض `.عام` و`.محلي`،
وتسلسلات هروب السلاسل المقبولة.
