# دليل لغة نَظْم 0.4

يوثق هذا الملف صيغة المصدر المنفذة حاليا. ليس وعدا بأن كل تركيب إكس86-64
(`x86-64`) مدعوم؛ قبول المحلل المعجمي والمحلل النحوي للتركيب لا يعني أن
المرمز يدعم كل توليفة معاملات.
عند غياب صيغة ترميز صريحة يجب أن يصدر نَظْم خطأ بدلا من تخمين بايتات.

## الملف والسطر

- الملف نص UTF-8 بامتداد `.نظم`.
- تنتهي كل عبارة مصدرية بنهاية السطر.
- يبدأ التعليق بـ`;` ويستمر إلى نهاية السطر.
- تقبل الفاصلة العربية `،` والفاصلة ASCII `,` بين المعاملات.
- الكلمات المفتاحية والوسوم حساسة لبايتات UTF-8؛ راجع
  [عقد يونيكود](UNICODE.md) و[المصطلحات المعتمدة](TERMINOLOGY.md).

مثال:

```text
.نص
.عام البداية

البداية:
    انقل سجل_المركم، ٦٠       ; نداء النظام: خروج
    خالف_بتيا فهرس_الوجهة، فهرس_الوجهة
    ناد_النظام
```

## السجلات

لا توجد صيغة رقمية مختصرة للسجلات في الإصدار 0.4. لكل سجل اسم عربي وصفي،
وتستخدم أجزاء العرض وأرقام السجلات العامة الممتدة الأرقام العربية-الهندية فقط.
تستنتج التعليمة عرض المعامل من اسم السجل؛ لا توجد لاحقة تعليمة لاتينية.

| سجل إكس86-64 | اسم نَظْم |
|---|---|
| `RAX` | `سجل_المركم` |
| `RCX` | `سجل_العداد` |
| `RDX` | `سجل_البيانات` |
| `RBX` | `سجل_القاعدة` |
| `RSP` | `مؤشر_المكدس` |
| `RBP` | `مؤشر_القاعدة` |
| `RSI` | `فهرس_المصدر` |
| `RDI` | `فهرس_الوجهة` |
| `R8`–`R15` | `سجل_عام_٨`–`سجل_عام_١٥` |
| `EAX`–`EDI` | الاسم المقابل مع اللاحقة `_٣٢`، مثل `سجل_المركم_٣٢` |
| `R8D`–`R15D` | `سجل_عام_٨_٣٢`–`سجل_عام_١٥_٣٢` |
| `AX`–`DI` | الاسم المقابل مع اللاحقة `_١٦`، مثل `سجل_المركم_١٦` |
| `R8W`–`R15W` | `سجل_عام_٨_١٦`–`سجل_عام_١٥_١٦` |
| `AL`، `CL`، `DL`، `BL`، `SPL`، `BPL`، `SIL`، `DIL` | الاسم المقابل مع اللاحقة `_٨` |
| `R8B`–`R15B` | `سجل_عام_٨_٨`–`سجل_عام_١٥_٨` |

لا تعرض اللغة سجلات البايت العليا `AH` و`CH` و`DH` و`BH`. ويجب أن يكون سجل
الأساس في معامل الذاكرة بعرض 64 بت حتى عندما يكون معامل القيمة أضيق.

## الأرقام والمعاملات

تقبل الأعداد العشرية أرقام ASCII والأرقام العربية-الهندية، وتقبل الصيغ
السداسية `0x...` والثنائية `0b...` والقيم السالبة حيث يسمح المعامل.

أشكال المعامل المنفذة:

```text
سجل_القاعدة              ; سجل
٤٢              ; قيمة فورية
اسم_وسم         ; مرجع وسم
[سجل_العداد]            ; ذاكرة عند سجل أساس
[سجل_العداد+٨]          ; سجل أساس مع إزاحة
[مؤشر_القاعدة-١٦]         ; سجل أساس مع إزاحة سالبة
"نص\n"          ; سلسلة مفكوكة الهروب داخل توجيه بيانات
```

لا تدعم صيغة الذاكرة الحالية سجلا فهرسيا أو معامل قياس أو أكثر من سجل. يجب
أن تمثل الإزاحة قيمة 32 بت موقعة، وإلا يرفضها المحلل النحوي.

تشمل التعليمات الصحيحة المنفذة النقل والحساب والمنطق والمقارنة والإزاحة
بعروض 8 و16 و32 و64 بت، إضافة إلى `وسع_بصفر` و`وسع_بإشارة`، والقسمة
`اقسم_موقع` و`اقسم_غير_موقع` و`وسع_إشارة_القسمة`، وتعليمات `عين_*` الاثنتي
عشرة التي تكتب نتيجة بعرض 8 بت. ويقبل `ناد` و`اقفز` هدفا غير مباشر في سجل
64 بت.

## الوسوم والرؤية

يعرّف النقطتان وسمًا عند الإزاحة الحالية:

```text
حلقة:
    انقص سجل_البيانات
    اقفز_غير_صفر حلقة
```

أسماء الوسوم في مصدر نظم عربية فقط. يكتب مدخل البرنامج العام ورمز الربط باسم
`الرئيسية` بلا تحويل إلى `main` أو أي اسم لاتيني. يحدد المستدعي هذا الرمز العربي
صراحة كنقطة الدخول عندما لا يستخدم غلاف تشغيل المنصة.

الوسم محلي افتراضيا. يمكن إعلان الرؤية قبل التعريف أو بعده:

```text
.عام البداية
.محلي مساعد
```

يجب أن يُعرّف كل وسم مذكور في `.عام` أو `.محلي` داخل الملف الحالي، ولا يجوز
جمع التوجيهين للوسم نفسه. يعلن `.خارجي اسم` رمزا عربيا عاما غير معرف في
الملف، وتنتج مراجع `ناد` و`اقفز` إليه ترحيل PC-relative صالحا لـ ELF64 أو
COFF. لا يحول نَظْم الرمز العربي إلى اسم منصة لاتيني.

## الأقسام وتوجيهات البيانات

| التوجيه | السلوك الحالي |
|---|---|
| `.نص` | الانتقال إلى قسم التعليمات `.text` |
| `.بيانات` | الانتقال إلى قسم البيانات `.data` |
| `.بيانات_للقراءة` | الانتقال إلى `.rodata` في إلف64 أو `.rdata` في كوف |
| `.غير_مهيأة` | الانتقال إلى `.bss`؛ يقبل الحجز الصفري والمحاذاة فقط |
| `.عدد٨` | عنصر بعرض 8 بت لكل معامل |
| `.عدد١٦` | عنصر بعرض 16 بت وترتيب البايت الأقل أهمية أولا |
| `.عدد٣٢` | عنصر بعرض 32 بت وترتيب البايت الأقل أهمية أولا |
| `.عدد٦٤` | عنصر بعرض 64 بت؛ يقبل قيمة أو اسم رمز عربي بترحيل مطلق |
| `.محاذاة N` | حشو القسم إلى حد بايتات `N`، ويجب أن يكون قوة للعدد 2 |
| `.مساحة_صفرية N` | إصدار `N` بايتات صفرية |
| `.سلسلة_منتهية_بصفر "..."` | إصدار UTF-8 المفكوك ثم بايت `NUL` |
| `.عام وسم` | جعل الرمز عاما في إلف64 أو كوف |
| `.محلي وسم` | تثبيت الرمز محليا |
| `.خارجي وسم` | إعلان رمز عربي غير معرف يحله الرابط |

توجيهات البيانات تصدر بايتات داخل `.بيانات`. الحد البنيوي الحالي ثلاثة
معاملات لكل عبارة مصدرية؛ قسم القوائم الأطول على أكثر من سطر.
تقبل توجيهات 8 و16 و32 بت المجال الموقع الكامل والمجال غير الموقع الكامل
للحجم نفسه، وترفض ما عداه بدلا من اقتطاعه. تقبل `.عدد٦٤` كل قيمة يمثلها
`int64_t` أو اسم رمز عربي محلي أو خارجي؛ يصدر الاسم صفرا مؤقتا وقيد ترحيل
`ABS64` في `.rela.data` لإلف64 أو جدول ترحيلات `.data` لكوف. يتطلب
`.مساحة_صفرية` قيمة واحدة بين الصفر و`INT_MAX`.
تقبل `.محاذاة` حدا صريحا بين 1 و4096 بايت؛ يختلف هذا عمدا عن
`.p2align` في GAS الذي يأخذ أسّ القوة، ولذلك يخفض باعث باء الأس إلى عدد
البايتات قبل كتابة مصدر نَظْم.

تسلسلات الهروب المنفذة في السلاسل هي `\n` و`\t` و`\r` و`\0` و`\"` و`\\`.

لا تحمل `.غير_مهيأة` بايتات في الملف الكائني: يصدرها إلف64 كقسم
`SHT_NOBITS` ويصدرها كوف كبيانات غير مهيأة. تبقى أحجام الرموز وإزاحاتها
حقيقية في جدول الرموز، ولا يسمح القسم بقيمة ابتدائية غير صفرية.

## التعليمات

الأسماء المعتمدة المنفذة في المحلل المعجمي:

- النقل: `انقل`، `ادفع`، `اسحب`، `احسب_عنوان`، `وسع_بصفر`، `وسع_بإشارة`.
- الحساب: `أضف`، `اطرح`، `اضرب_موقع`، `اقسم_موقع`، `اقسم_غير_موقع`، `وسع_إشارة_القسمة`، `زد`، `انقص`، `اعكس_الإشارة`.
- المنطق والإزاحة: `و_بتيا`، `أو_بتيا`، `خالف_بتيا`، `اعكس_البتات`، `ازح_يسارا`، `ازح_منطقيا_يمينا`، `ازح_حسابيا_يمينا`.
- المقارنة: `قارن`، `اختبر_البتات`، وعائلة `عين_*` ذات الاثني عشر شرطا.
- التحكم: `اقفز`، `ناد`، `ارجع`.
- القفز الشرطي: `اقفز_مساو`، `اقفز_غير_مساو`، `اقفز_أكبر`،
  `اقفز_أكبر_أو_مساو`، `اقفز_أصغر`، `اقفز_أصغر_أو_مساو`، `اقفز_صفر`،
  `اقفز_غير_صفر`، `اقفز_سالب`، `اقفز_غير_سالب`.
- النظام: `ناد_النظام`، `لا_تفعل`، `أوقف`، `اطلب_مقاطعة`.

جدول الصيغ الفعلي في `src/encoder/table.c` هو مرجع توليفات المعاملات
المنفذة. لا تفترض أن وجود اسم التعليمة يعني دعم كل صيغة إكس86-64 لها.

## أخطاء شائعة

1. **تهجئة قديمة:** لا تقبل أسماء 0.3 القديمة. يرفضها نَظْم ويذكر بديل 0.4
   الدقيق دون تنفيذها كأسماء بديلة.
2. **اسم سجل مختصر:** استخدم الاسم الوصفي الكامل مثل `فهرس_الوجهة` أو
   `سجل_عام_٨`. لا تقبل الصيغ الرقمية المختصرة أو الأرقام اللاتينية في الاسم.
3. **نسيان الفاصلة:** `انقل سجل_المركم سجل_العداد` خطأ؛ استخدم `انقل سجل_المركم، سجل_العداد` أو الفاصلة
   الإنجليزية.
4. **عنوان ذاكرة غير مدعوم:** `[سجل_العداد+سجل_البيانات]` و`[سجل_العداد*4]` غير منفذين. استخدم سجل
   أساس وإزاحة عددية فقط.
5. **قيمة لا تمثلها الصيغة:** لا يقتطع نَظْم فورية أو قيم توجيهات البيانات
   أو إزاحة أو `rel32` بصمت؛ صغر القيمة أو اختر صيغة مدعومة.
6. **وسم مكرر أو غير محلول:** كل تعريف فريد، وكل مرجع يجب أن يجد تعريفا صالحا
   في القسم المناسب.
7. **رؤية متعارضة:** لا تجمع `.عام` و`.محلي` للوسم نفسه، ولا تعلن رؤية وسم
   يبقى بلا تعريف.
8. **سلسلة غير مغلقة أو تسلسل هروب مجهول:** أغلق `"` واستخدم تسلسلات الهروب
   المذكورة فقط.
9. **صيغة معاملات غير مدعومة:** ظهور الاسم في قائمة التعليمات لا يبرر توليفة
   معاملات جديدة؛ راجع الاختبارات أو شغّل نَظْم وانتظر خطأ صريحا.

أمثلة تشخيص قابلة للتشغيل موجودة في
[`examples/diagnostics/`](../examples/diagnostics/).
