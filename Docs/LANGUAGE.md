# دليل لغة نَظْم 0.3

يوثق هذا الملف صيغة المصدر المنفذة حاليا. ليس وعدا بأن كل تركيب x86-64
مدعوم؛ قبول lexer وparser للتركيب لا يعني أن encoder يدعم كل توليفة معاملات.
عند غياب صيغة ترميز صريحة يجب أن يصدر نَظْم خطأ بدلا من تخمين بايتات.

## الملف والسطر

- الملف نص UTF-8 بامتداد `.مجمع`.
- ينتهي كل statement بنهاية السطر.
- يبدأ التعليق بـ`;` ويستمر إلى نهاية السطر.
- تقبل الفاصلة العربية `،` والفاصلة ASCII `,` بين المعاملات.
- الكلمات المفتاحية والوسوم حساسة لبایتات UTF-8؛ راجع
  [عقد Unicode](UNICODE.md).

مثال:

```text
.نص
.عام البداية

البداية:
    احمل ر0، ٦٠       ; syscall: exit
    خالف ر7، ر7
    نداء_نظام
```

## السجلات

الصيغة الرقمية هي `ر0` إلى `ر15`. الجزء الرقمي في اسم السجل ASCII فقط؛
`ر7` سجل، أما `ر٧` فليس alias له في الإصدار 0.3.

توجد الأسماء التالية:

| الاسم | السجل |
|---|---|
| `مجمع` | `ر0` / `rax` |
| `قاعدة_ب` | `ر1` / `rbx` |
| `عداد` | `ر2` / `rcx` |
| `بيانات` | `ر3` / `rdx` |
| `مكدس` | `ر4` / `rsp` |
| `قاعدة` | `ر5` / `rbp` |
| `مصدر` | `ر6` / `rsi` |
| `وجهة` | `ر7` / `rdi` |

## الأرقام والمعاملات

تقبل الأعداد العشرية أرقام ASCII والأرقام العربية-الهندية، وتقبل الصيغ
السداسية `0x...` والثنائية `0b...` والقيم السالبة حيث يسمح المعامل.

أشكال المعامل المنفذة:

```text
ر3              ; سجل
٤٢              ; قيمة فورية
اسم_وسم         ; مرجع وسم
[ر1]            ; ذاكرة عند سجل أساس
[ر1+٨]          ; سجل أساس مع إزاحة
[ر5-١٦]         ; سجل أساس مع إزاحة سالبة
"نص\n"          ; سلسلة مفكوكة الهروب داخل توجيه بيانات
```

لا تدعم صيغة الذاكرة الحالية index أو scale أو أكثر من سجل. يجب أن تمثل
الإزاحة قيمة signed 32-bit، وإلا يرفضها parser.

## الوسوم والرؤية

يعرّف النقطتان وسمًا عند الإزاحة الحالية:

```text
حلقة:
    انقص ر2
    اقفز_لاصفر حلقة
```

الوسم محلي افتراضيا. يمكن إعلان الرؤية قبل التعريف أو بعده:

```text
.عام البداية
.محلي مساعد
```

يجب أن يُعرّف كل وسم مذكور في `.عام` أو `.محلي` داخل الملف الحالي، ولا يجوز
جمع التوجيهين للوسم نفسه. الرموز الخارجية وrelocations النداء/القفز الخارجية
غير منفذة بعد.

## الأقسام وتوجيهات البيانات

| التوجيه | السلوك الحالي |
|---|---|
| `.نص` | الانتقال إلى قسم التعليمات `.text` |
| `.بيانات` | الانتقال إلى قسم البيانات `.data` |
| `.بايت` | عنصر 8-bit لكل معامل |
| `.عدد١٦` | عنصر little-endian بحجم 16-bit |
| `.عدد٣٢` | عنصر little-endian بحجم 32-bit |
| `.عدد٦٤` | عنصر little-endian بحجم 64-bit |
| `.مساحة N` | إصدار `N` بايتات صفرية |
| `.سلسلة "..."` | إصدار UTF-8 المفكوك ثم بايت `NUL` |
| `.عام وسم` | جعل الرمز عاما في ELF64/COFF |
| `.محلي وسم` | تثبيت الرمز محليا |

توجيهات البيانات تصدر بايتات داخل `.بيانات`. الحد البنيوي الحالي ثلاثة
معاملات لكل statement؛ قسّم القوائم الأطول على أكثر من سطر.

تسلسلات الهروب المنفذة في السلاسل هي `\n` و`\t` و`\r` و`\0` و`\"` و`\\`.

## التعليمات

الأسماء canonical المنفذة في lexer:

- النقل: `احمل`، `ادفع`، `اسحب`، `عنون`.
- الحساب: `أضف`، `اطرح`، `اضرب`، `اقسم`، `زد`، `انقص`، `اسلب`.
- المنطق والإزاحة: `و`، `أو`، `خالف`، `انفِ`، `ازحل`، `ازحي`، `ازحر`.
- المقارنة: `قارن`، `اختبر`.
- التحكم: `اقفز`، `نادِ`، `ارجع`.
- القفز الشرطي: `اقفز_مساوٍ`، `اقفز_مختلف`، `اقفز_أكبر`،
  `اقفز_أكبر_أو`، `اقفز_أصغر`، `اقفز_أصغر_أو`، `اقفز_صفر`،
  `اقفز_لاصفر`، `اقفز_سالب`، `اقفز_موجب`.
- النظام: `نداء_نظام`، `لاشيء`، `أوقف`، `قاطع`.

جدول الصيغ الفعلي في `src/encoder/table.c` هو مرجع توليفات المعاملات
المنفذة. لا تفترض أن وجود اسم التعليمة يعني دعم كل صيغة x86-64 لها.

## أخطاء شائعة

1. **إسقاط الحركة:** `ناد` ليس `نادِ`، و`انف` ليس `انفِ`. لا يطبّع الإصدار
   0.3 الكلمات ولا يضيف aliases.
2. **أرقام عربية في اسم السجل:** استخدم `ر7` لا `ر٧`. الأعداد الفورية تقبل
   العربية-الهندية، لكن اسم السجل لا يقبلها.
3. **نسيان الفاصلة:** `احمل ر0 ر1` خطأ؛ استخدم `احمل ر0، ر1` أو الفاصلة
   الإنجليزية.
4. **عنوان ذاكرة غير مدعوم:** `[ر1+ر2]` و`[ر1*4]` غير منفذين. استخدم سجل
   أساس وإزاحة عددية فقط.
5. **قيمة لا تمثلها الصيغة:** لا يقتطع نَظْم فورية أو displacement أو
   `rel32` بصمت؛ صغّر القيمة أو اختر صيغة مدعومة.
6. **وسم مكرر أو غير محلول:** كل تعريف فريد، وكل مرجع يجب أن يجد تعريفا صالحا
   في القسم المناسب.
7. **رؤية متعارضة:** لا تجمع `.عام` و`.محلي` للوسم نفسه، ولا تعلن رؤية وسم
   يبقى بلا تعريف.
8. **سلسلة غير مغلقة أو escape مجهول:** أغلق `"` واستخدم تسلسلات الهروب
   المذكورة فقط.
9. **صيغة معاملات غير مدعومة:** ظهور الاسم في قائمة التعليمات لا يبرر توليفة
   معاملات جديدة؛ راجع الاختبارات أو شغّل نَظْم وانتظر خطأ صريحا.

أمثلة تشخيص قابلة للتشغيل موجودة في
[`examples/diagnostics/`](../examples/diagnostics/).
